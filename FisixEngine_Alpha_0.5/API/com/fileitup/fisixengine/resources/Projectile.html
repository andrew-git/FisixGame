A class for the quick creation of projectiles/bullets. Provides methods to control the projectile's orientation and speed, as well as methods to find out when the projectile collided, and provides data about the collision.

Property detail
removeOnCollisionproperty
public var removeOnCollision:Boolean

Define this function in your code to perform and action once the projectile collides. As parameters, this function takes:
A CollisionObject (the object that was hit)
A Vector representing the point of collision (in global space).


Example
The following code creates a Projectile and uses the onHit() method. 
   var missile:Projectile = new Projectile(600,0,1,true)
   missile.onHit = function(){
    //play animation explosion
   }
   missile.shootAngle(60,20) //60 degrees, speed of 20

removeOnCollisionproperty 
public var removeOnCollision:Boolean

If true, the projectile will automatically be removed when it collides with an object. The onHit() method will be called whether this value is true or not.
